Структура файла

Файл настроек, это простой «ini» файл с двумя секциями

 1[one_C]
 2queryNumberX = /unf_vz_const/hs/test_s/V1/GetProductRemains
 3lquery = /unf_vz_const/hs/
 4
 5shopquery = /unf_vz_const/hs/GetChangeShop/V1/GetShop
 6workshift = /unf_vz_const/hs/Workshift/V1/post_workshop
 7workshift_open = /unf_vz_const/hs/Workshift/V1/post_workshop_open
 8
 9cat_skl = \\192.168.252.250\unf_teleport
10server_ip = 192.168.252.250
11port = 80
12
13[artix]
14path = world
15tPriceQR = 1
16server_ip = 10.0.0.239
17exchange_cat = //10.0.0.239/obmen/dict/
18enablepricemanual = 1
19database = documentsAll
20user= netroot
21passwd= netroot
Значение параметров¶

Параметр

Значение

queryNumberX

Запрос для получения остатков по магазину

lquery

Шаблон запроса

shopquery

Получить номера магазинов с изменениями

workshift

Отправить закрытые смены

workshift_open

Отправить открытые смены

cat_skl

Каталог обмена

server_ip

Адрес сервера УНФ

port

Порт сервера УНФ

server_ip

Адрес кассового сервера

exchange_cat

Каталог обмена кассового сервера

database

База в БД кассового сервера

user

Пользователь MySQL

passwd

Пароль пользователя MySQL
