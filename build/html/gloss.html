aif¶

Artix Interchange (AIF) – основной формат обмена семейства продуктов Artix. Данный формат поддерживает полный набор функциональных возможностей кассового узла. Для остальных форматов поддержаны только базовые возможности программы. Перечень поддерживаемых возможностей ограничен самими форматами и их совместимостью с функционалом Artix.
