Внешняя программма¶

Внешняя программа реализована на языке Python и расположена на отдельном сервере с установленной серверной операционной системой Linux Ubuntu 22.04.2 LTS (GNU/Linux 5.15.0-69-generic x86_64)

Программа состоит из двух блоков, первый отвечает за получение остатков товаров из обработки и передачи на кассовый сервер Artix, второй за отслеживание открытия и закрытия смен на кассах и переодической передачи этих данных в УНФ.
